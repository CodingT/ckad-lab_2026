## Question 17 – Retrieve Pod Logs and Identify High CPU Usage

This question has two tasks:

**Task 1 – Retrieve Pod Logs**
- Deploy pod `winter` from manifest at `/root/winter.yaml`
- Retrieve all application logs from the running pod
- Store logs in file `/root/log_Output.txt`

**Task 2 – Find Pod with Highest CPU Usage**
- From pods running in namespace `cpu-stress`
- Identify the pod consuming the most CPU
- Write only the pod name to file `/root/pod.txt`
